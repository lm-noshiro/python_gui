# Cropper

## Usage

1. create venv

```sh
$ python3 -m venv env
$ source env/bin/activate
```

2. Install dependencies

```sh
(env) $ pip install -r requirements.txt
```

3. Run `cropper.py` or `cropper_rx.py`

```sh
(env) $ python3 ./src/cropper.py
```

or

```sh
(env) $ python3 ./src/cropper_rx_ver.py
```
